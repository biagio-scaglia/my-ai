# Consigli di Programmazione di Coddy

## Progetti Consigliati

1.  **To-Do List CLI**: Un classico per iniziare. Impara a gestire input, file e classi.
2.  **Web Scraper di News**: Usa `requests` e `BeautifulSoup` per raccogliere notizie tech.
3.  **Bot Telegram/Discord**: Impara le API asincrone.
4.  **Portfolio Personale**: Usa HTML/CSS o un generatore statico.

## Stack Moderno Python

- **Web Framework**: FastAPI (veloce, moderno) o Django (robusto).
- **Data Science**: Pandas, Polars (molto veloce).
- **AI/ML**: PyTorch, Hugging Face Transformers.
- **Testing**: Pytest.
- **Linting/Formatting**: Ruff (velocissimo).

## Best Practices

- Usa sempre i **Virtual Environment** (`venv` o `uv`).
- Scrivi **Type Hints** nel codice Python.
- Documenta le funzioni con **Docstrings**.
