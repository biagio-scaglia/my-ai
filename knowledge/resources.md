# Risorse e Documentazione per lo Stack di Biagio

Ecco una lista curata di documentazione ufficiale e risorse avanzate per le tecnologie che utilizzi.

## Linguaggi di Programmazione

### Kotlin & Java

- **Kotlin Docs**: [kotlinlang.org/docs](https://kotlinlang.org/docs/home.html) - Il riferimento assoluto.
- **Java Platform SE Docs**: [docs.oracle.com/en/java/](https://docs.oracle.com/en/java/)

### Python

- **Python 3 Documentation**: [docs.python.org/3/](https://docs.python.org/3/)
- **Real Python**: Ottimo per tutorial avanzati e best practices.

### JavaScript & TypeScript

- **MDN Web Docs**: [developer.mozilla.org](https://developer.mozilla.org/) - La Bibbia per JS/HTML/CSS.
