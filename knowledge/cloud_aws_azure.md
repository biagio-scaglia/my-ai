# Cloud Computing & Infrastructure (AWS/Azure)

## Concetti Base

- **IaaS**: VM nude e crude (EC2, Azure VM).
- **PaaS**: Piattaforme gestite (Elastic Beanstalk, App Service).
- **FaaS**: Serverless (AWS Lambda, Azure Functions).

## AWS (Amazon Web Services)

- **S3**: Storage oggetti infinito.
- **EC2**: Server virtuali.
- **RDS**: Database gestiti (Postgres, MySQL).
- **IAM**: Gestione identità e permessi (Cruciale!).

## Azure

- **Blob Storage**: Equivalente S3.
- **Azure DevOps**: Suite CI/CD integrata.
- **Cosmos DB**: Database NoSQL globale.

## Docker & Kubernetes (K8s)

- **Container**: Pacchettizza app e dipendenze.
- **Pod**: Unità minima K8s.
- **Service**: Espone un set di Pod.
- **Deployment**: Gestisce repliche e aggiornamenti rolling.
