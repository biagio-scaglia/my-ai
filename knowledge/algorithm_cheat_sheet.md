# Algorithms & Data Structures Cheat Sheet

## Notazione Big O

- **O(1)**: Costante (Hash Map lookup).
- **O(log n)**: Logaritmica (Binary Search).
- **O(n)**: Lineare (Scorrere una lista).
- **O(n log n)**: Ordinamento efficiente (Merge Sort, Quick Sort).
- **O(n^2)**: Quadratico (Loop annidati, Bubble Sort).

## Strutture Dati Comuni

- **Array**: Accesso veloce O(1), inserimento lento O(n).
- **Linked List**: Inserimento veloce O(1), accesso lento O(n).
- **Hash Map (Dictionary)**: Chiave-Valore. Accesso/Ins mediamente O(1).
- **Stack**: LIFO (Last In First Out).
- **Queue**: FIFO (First In First Out).
- **Tree/Graph**: Relazioni gerarchiche o reti.

## Algoritmi Fondamentali

- **Binary Search**: Dividi et impera su array ordinati.
- **BFS (Breadth First)**: Esplorazione a livelli (cammino minimo grafi).
